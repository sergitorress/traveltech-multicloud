1 CREAR GITHUB

Necesitas un repositorio en GitHub para tu código.

  1. Abre: https://github.com
  2. Click en "Sign up" o "Sign in"
  3. Usuario: micronigga
  4. Completa el registro
⚠️ Nota: Una vez registrado, crear un nuevo repositorio:
  • Nombre: traveltech-multicloud
  • Selecciona: PUBLIC
  • Click: Create repository

2 SUBIR CÓDIGO A GITHUB

Copia tu código a GitHub desde tu PC.

Abre PowerShell en la carpeta del proyecto y ejecuta:

git push -u origin main

Si pide autenticación:

  1. Ir a: https://github.com/settings/tokens
  2. Click: Generate new token
  3. Dar permisos: repo
  4. Copiar el token que genera
  5. En la terminal, cuando pida contraseña, pegar el token

3 DESPLEGAR EN RENDER

Desplegar 3 servicios en Render (Frontend + 2 Microservicios).

A) Frontend:

  1. Abre: https://render.com
  2. Sign up con GitHub (Connect with GitHub)
  3. Click: New + → Web Service
  4. Conecta tu repositorio traveltech-multicloud
  5. Configuración:
    • Name: traveltech-frontend
    • Root: frontend
    • Build: npm install
    • Start: npm start
    • Plan: Free
  6. Click: Create Web Service
  7. Espera 10 minutos → Copia la URL que aparece

B) Wishlist (repetir A con):

Name: traveltech-wishlist Root: service-wishlist Build: npm install Start: npm start

C) Comments (repetir A con):

Name: traveltech-comments Root: service-comments Build: npm install Start: npm start

Guarda las 3 URLs que obtendrás:

4 ACTUALIZAR URLs EN EL CÓDIGO

Conecta tu Frontend con los microservicios desplegados.

  1. En VS Code, abre: frontend/public/script.js
  2. Busca (Ctrl+F): const SERVICES = {
  3. Reemplaza esto:
    const SERVICES = { FAVORITES: 'http://localhost:3001', WISHLIST: 'http://localhost:3002', COMMENTS: 'http://localhost:3003' };
  4. Con las URLs de Render que copiaste:
    const SERVICES = { FAVORITES: 'https://traveltech-wishlist.onrender.com', WISHLIST: 'https://traveltech-wishlist.onrender.com', COMMENTS: 'https://traveltech-comments.onrender.com' };
  5. Guarda (Ctrl+S)
  6. En PowerShell, ejecuta:
    git add frontend/public/script.js git commit -m "Update production URLs" git push
