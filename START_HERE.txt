🔧 TECNOLOGIES UTILITZADES
───────────────────────────────────────────────────────────────

Frontend (Port 3000):
  • HTML5 + CSS3 + JavaScript vanilla
  • Express.js (static server)
  • FETCH API (comunicació)

Microserveis:
  • Node.js + Express.js
  • REST API (JSON)
  • In-memory database
  • UUID para comentaris
  • CORS enabled

Cloud Ready:
  • Render, Railway, Fly.io
  • Dockerfile-ready
  • Environment variables
  • Health checks
